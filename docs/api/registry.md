# deepecgkit.registry

Decorator-based registration system for models and datasets. Registered components are
automatically discoverable by the CLI and other consumers.

::: deepecgkit.registry
    options:
      members:
        - register_model
        - register_dataset
        - get_model
        - get_dataset
        - get_model_names
        - get_dataset_names
        - get_model_info
        - get_dataset_info
